# 手動テスト仕様書

自動テスト（`node test_flow.js`）で確認できないExcel内容・見た目・操作感を中心に確認する。

## テスト準備

1. DBを完全リセットする: アプリ停止 → `Zaiko/app.db` を削除 → `dotnet ef database update --project Zaiko` → 再起動
2. アプリを起動（`dotnet run --project Zaiko`）
3. `http://localhost:5038` でログイン（admin / admin）

> セクション0（SeedDataのテスト）はリセット後に単独で実施する。SeedDataを投入するとセクション2〜5で手動作成するデータと衝突するため、セクション0の後にDBを再リセットしてからセクション1以降を進める。
